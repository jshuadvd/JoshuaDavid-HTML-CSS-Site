Experience With Programming

Some time ago, I needed a website for one of my businesses and had no idea where to begin. I found an online template, but I didn't want to use the template the way it was. I opened it up in Dreamweaver and began moving things around. Soon enough, I completed the project and my friends liked it so much that they asked me to do the same for them.

After doing this a few times I started getting the hang of HTML and CSS. After that I started playing around in Flash and Wordpress and eventually became able to hack around and manipulate the code to do what I wanted it to. And if I didn't know how to do something, I would search around online until I found an answer. I'm not at a point where I can write code completely from scratch, but I can interpret it, for the most part, and have a basic understanding of the fundamental concepts. I have also built some applications by following along with online tutorials, which you can view on my GitHub.

Tools And Languages

To build this site I created a mock-up in Photoshop and then hand coded everything in Coda 2. I usually use Sublime Text 3, but I had Coda and have never really used it before, so I wanted to try it out. The site only consists of HTML and CSS as you will be able to see when you view the code. I used Firefox and Firebug to preview and debug and for version control I am using Github. I am also using iTerm for the command line to commit my repositories. I usually use my iPad and iPhone to check responsive features, but I haven't had time to do the media queries at the point of writing this. I hope that I am able to get to it before the submission deadline. Lastly, I used stack overflow because I ran into a problem with div positioning while creating the hexagon hovers.

I would have liked to make the site more dynamic, but you mentioned that it needed to be hand coded and would have had to pull from this great CSS3 library I like to use located here.
